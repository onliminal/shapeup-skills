# New Product Example

This example shows the longest path through the skill:

`Explore -> Evidence -> Frame -> Shape -> De-risk -> Pitch`

## Scenario

A founder wants to build a lightweight lead-management product for home-service owner-operators who lose inbound leads across text, email, voicemails, and social DMs.

## What this example teaches

- how Explore stays broad before narrowing to a wedge
- how Evidence turns a promising wedge into a grounded problem
- how the skill moves from opportunity-space language into a framed problem with appetite
- what a complete set of finished artifacts looks like
