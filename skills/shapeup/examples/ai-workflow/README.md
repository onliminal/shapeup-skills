# AI Workflow Example

This example shows an AI/workflow concept moving through the full upstream path:

`Explore -> Evidence -> Frame -> Shape -> De-risk -> Pitch`

## Scenario

A team wants an AI workflow to turn messy escalation context into a clean engineering handoff brief. The example shows how to keep the conversation grounded in a specific operational pain rather than a vague "AI assistant" idea.

## What this example teaches

- how Explore handles an AI-flavored idea without jumping straight to solution design
- how Evidence reduces "this sounds useful" uncertainty before appetite
- how to shape an AI workflow around human review and explicit boundaries
