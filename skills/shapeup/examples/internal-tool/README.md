# Internal Tool Example

This example shows a shorter, more direct planning path:

`Frame -> Shape -> De-risk -> Pitch`

## Scenario

A finance operations team needs an internal queue for billing-import exceptions that are currently triaged through Slack, spreadsheets, and memory.

## What this example teaches

- when it is appropriate to skip Explore and Evidence
- how to keep an internal-tool framing conversation anchored to one operational pain
- how appetite and scope discipline work when the users are known and reachable
